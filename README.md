# Flask-Demo

## Description
Flask-Demo is a simple web application built using the Flask framework. It serves as a demonstration of basic Flask functionalities, including routing, templates, and static file handling.

## Features
- User-friendly interface
- Dynamic content rendering using Jinja2 templates
- Static file serving for CSS and JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Jeremiah-Jefry/Flask-Demo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Flask-Demo
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```bash
python app.py
```

Visit `http://127.0.0.1:5000` in your web browser to view the application.
